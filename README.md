



💬 ChatTranslatorApp

ChatTranslatorApp, kullanıcıların metinleri farklı diller arasında çevirebilmesini sağlayan bir C# Windows Forms masaüstü uygulamasıdır.
Uygulama, sade arayüzü ve hızlı çeviri mantığıyla temel bir dil çeviri çözümü sunar.

🚀 Projenin Amacı

Metinlerin farklı dillere çevrilmesini sağlamak

Masaüstü uygulamalarda çeviri sistemlerinin nasıl çalıştığını göstermek

API / servis tabanlı çeviri mantığını öğrenmek

🛠 Kullanılan Teknolojiler

C#

.NET Framework

Windows Forms

DevExpress UI Bileşenleri

Visual Studio
